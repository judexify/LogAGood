# LogAGood

LogAGood is a logistics management platform that connects customers, riders, and administrators in one application.

The project is designed to simplify the delivery process from order creation to successful delivery while providing each user with a dashboard tailored to their role.

## Users

Customer

Rider

Administrator

## Features

Create and manage delivery requests

Assign riders to deliveries

Track delivery progress

Manage customers and riders

Role based access control

Priority deliveries

Favourite riders

Delivery analytics

## Tech Stack

Next.js

TypeScript

Tailwind CSS

shadcn/ui

Supabase

React Hook Form

## Status

Currently under development ,inspired by the logistics operations of local delivery companies in Ilorin, Kwara State.
